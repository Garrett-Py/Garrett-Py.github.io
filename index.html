Projects (This section will probably be removed or replaced)

Encryption in Chaos

During university, I did an independent study in Intermediate Differential Equations. We used the book "Nonlinear Dynamics and Chaos" by Strogatz. In the book, there was a problem which mentioned that signals could be encrypted using the Lorenz equations, so as a capstone project, I was challenged with writing a program to encrypt a signal. The image above is the result, where the encrypted signal looks nothing like the original signal.

After I completed the course I went back and modified the code to encrypt audio signals. The challenge with this is picking the correct parameters for the Lorenz equations, since if you pick the wrong parameters, the encrypted and decrypted audio will sound the same. I was successfully able to find values for the parameters that would make the encrypted signal sound like radio static, and the decrypted signal sounded similar to the original one.

Modal Analysis

In my acoustics class there was a project where we did modal analysis on a sword. We theoretically calculated the modal frequencies of a sword, used COMSOL to do finite element analysis to calculate the modal frequencies, and also physically recorded the modal frequencies of an actual sword.

The modal frequencies and wave types (transverse, longitudinal, torsional) were compared to simulation and theory. The results can be found here.

RSA Cryptography Application

I built an application using Tkinter in Python to encrypt and decrypt messages using RSA cryptography. The application allows you to:

1. Generate public/priavte keys.

2. Encrypt messeges using a public key.

3. Decrypt messeges using a public and private key.

In the future, I plan to expand on this project and implement Elliptic Curve Cryptography into the application. The "Encryption in Chaos" project will also be rewritten in Python and added to this application.
